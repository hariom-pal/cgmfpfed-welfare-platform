# CGMFPFED Welfare Platform

## Overview

CGMFPFED Welfare Platform is an enterprise-grade Laravel application for managing welfare schemes such as Scholarship and Beema.

## Modules

- Common Platform
- Scholarship
- Beema
- Workflow Engine
- Dashboard
- Audit Trail
- CSC Connect Integration
- Wallet Integration

## Technology Stack

- Laravel 12
- PHP 8.4
- MySQL 8
- Bootstrap 5
- jQuery
- AJAX
